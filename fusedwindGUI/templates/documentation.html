Documentation

This page provides links to the documentation on all the different system models developed and integrated into the FUSED-Wind framework.

Overall Documentation:

  • FUSED-Wind (http://fusedwind.org for documentation and tutorials) is a partnership between DTU Wind Energy, NREL and others
  • The OpenMDAO modeling environment (http://openmdao.org) underlies FUSED-Wind
  • All the above software is built in the Python programming language (https://www.python.org/ for documentation and tutorials)

WISDEM documentation:

  • Basic WISDEM documentation for integrated assemblies can be found here: http://wisdem.github.io/WISDEM/
